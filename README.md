# project_excel

Стек: Excel: сводные таблицы, сводные диаграммы.

Целью проекта было проанализировать датасет с Kaggle про музыкальные тренды 2024, основанные на площадке Spotify. Анализ был сделан при помощи сводных таблиц, на основе которых были построены сводные диаграммы и выведен отдельный лист, где был составлен дашборд.

Шаги по достижению цели:

1. Преобразование исходных данных.

Загрузил датасет формата CSV, преобразовал его в формат xlsx. Нашел дубликаты и пропущенные значения. Заменил пропущенные значения на 0. Преобразовал текстовые форматы в числовой там, где это необходимо. В том числе преобразовал американский формат даты к европейскому (единому для всех дат). Данный этап выполнялся на листе "Очищенные данные".

2. Создание сводных таблиц.

Создал сводные таблицы, выполняющие ряд аналитических расчетов, а именно: 
- топ 5 артистов по количеству просмотров стримов на Spotify;
- cредняя популярность трека (Spotify Popularity) в разбивке по наличию эксплицитного контента (Explicit Track);
- количество треков и средний Track Score по месяцам выпуска (Release Date);
- Сравнение суммарных просмотров на YouTube и суммарных стримов на Spotify.

Все сводные таблицы находятся на листе "Сводный анализ", на листе "Корелляция" приведена таблица для создания точечного графика (для сравнения суммарных просмоторов на YouTube и стримов на Spotify). 

3. Создал отдельный лист "Рейтинг артистов".

В листе создал таблицу, где при помощи функций МАКС, ЕСЛИ, ПОИСКПОЗ, ИНДЕКС составил рейтингов артистов, которые входили в топ-100 артистов по кол-ву просмотров стримов Spotify. Помимо прочего, таблица отвечала на вопрос сколько песен артистов входят в топ-100 песен Spotify, сколько просмоторов на самом популярном треке, его номер в таблице и название этого трека с именем артиста.

4. Составил дашборд на основе сводных диаграмм.

На основе полученных результатов, составил дашборд на отдельном листе "Дашборд", где приведены 4 графика с аналитическими срезами. 

5. Подготовил выводы о проекте.

Выводы о полученных результатах приведены на отдельном листе "Выводы".
